# Geo Distributed API Platform

Reference architecture and implementation examples for building globally distributed APIs with regional routing, failover, traffic steering, and compliance-aware deployments.

## Features

- Geo-based traffic routing
- Multi-region API deployment
- Active-active architecture
- Regional failover
- DNS-based routing
- HAProxy examples
- Nginx examples
- Kubernetes deployment patterns

## Architecture

Client -> GeoDNS -> Regional HAProxy -> Nginx -> API Nodes

## Examples

- Nginx geo routing
- HAProxy failover
- GeoDNS
- Kubernetes multi-region
